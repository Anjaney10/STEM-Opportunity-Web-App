# sci-intern-db

**Global Scholarships & Internship Opportunities for STEM Students**

A beautiful, interactive web app built with **Streamlit** that helps STEM students worldwide discover PhD scholarships, fellowships, and research internships. Features an interactive world map, live data scraping, smart filters, and detailed opportunity cards.

---

## Features

- **Interactive World Map** powered by Plotly
- **Smart Search & Filters** (by type, keywords)
- **Live Data** from public sources + curated premium opportunities
- **Clean, Responsive Dashboard** with detailed view
- **Fast Caching** for better performance
- **One-click Refresh** for latest opportunities

##  Opportunity Types Covered
- PhD Scholarships
- Research Fellowships
- Summer/Winter Internships
- Fully-funded Programs

Popular programs included: **NSF GRFP**, **DAAD RISE**, **Fulbright**, **Google PhD Fellowship**, **CERN**, etc.

---

##  How to Run Locally

```bash
git clone https://github.com/Anjaney10/sci-intern-db.git
cd sci-intern-db

pip install -r requirements.txt

streamlit run app.py
```
